Elementary OS使用體驗

Mar 13, 2016


系統信息
elementary OS 0.3.2 Freya (64-bit)基于 Ubuntu 14.04 构建

硬件信息
ThinkPad x201
處理器：四核 Intel® Core™ i5 CPU M 540 @ 2.53GHz
內存：4GB（實際可用3.7GB）
顯卡：Intel Corporation Core Processor Integrated Graphics Controller (rev 02)
硬盤：朗科128G固態硬盤

　　使用elementary OS已經不是第一次，剛接觸linux系統的時候，百度了挺久，最後選擇了大多數人都推薦的ubuntu，但是由於界面無法忍受，外加當時對linux沒什麼概念，安裝了沒多久就刪了。之後的後場一段時間都沒有再碰linux。
　　後來在一次逛軟件網站的時候，偶然發現了eOS，一下子被它的標題吸引了：Elementary OS 中文版 - 号称最漂亮的 Linux 系统！Ps：我在找這個網址的時候突然發現微軟的Visual Studio Code竟然在linux下也能用，當初只是有瞭解到VSC免費開源了，但那時候沒注意到跨平臺，現在想想我折騰編輯器到底在幹嘛，有這 等神器不用，畢竟在windows下已經習慣了。回到正題，作爲顏值黨，看到最漂亮立馬就被吸引了，然後看了下簡介，還有介紹視頻，覺得不錯，於是又撿起linux開始折騰了，當然那時候折騰也主要在系統的美化方面，裝主題，改圖標，改頂欄透明度什麼的，其實就是爲閒的，或者說爲了顯示自己的特例獨行，而且那時候windows下的美化已經被我折騰得差不多了，所以就像嘗試下新鮮的事物。於是當天我就把系統下載下來開始折騰了。
　　當時的eOS系統還是Luna，而不是現在的Freya。Luna折騰幾次後我也放棄過，之後Freya出來後我又折騰過，總之前前後後一共折騰了有五六次，直到現在。
　　迴歸正題中的整體，談談這段時間的使用感受，這小黑是我專門從鹹魚上淘來來折騰系統的，到手第二天就買了塊固態換上了。固態還是新的，流暢度暫時沒問題，就是不知道這固態經不經得住考驗，當然這和系統關係不大。因爲eOS基於ubuntu，所以應用相對來說還是不缺的，和windows就另當別論了。ubuntu的教程百度谷歌也較多，出了問題查詢比較方便。eOS從Luna以來bug從來沒斷過，這是不可避免的，雖然一直有問題，但我還是愛eOS。現在的0.3.2版本我也遇到了不少bug，比如桌面壁紙都無法自定義……這真的是有點過分了吧……比如終端第一個窗口無法在中dock顯示，要打開第二個新窗口纔會顯示兩個窗口，再比如有時候切換窗口會閃屏，畫面撕裂，這可能是電腦配置低的原因也說不定，還不確定是不是bug。
　　雖然有bug但對使用體驗影響不大，現在除了沒QQ不是怎麼方便，其他沒什麼大問題，雖然討厭QQ，但是沒有確實不方便，至少傳文件QQ方便太多。微信有網頁版，chrome應用商店也有微信，問題不大。上上網，看看在線視頻，寫寫博客已經足夠可，電腦也夠用了，反正玩不了遊戲，至於辦公有WPS，不過我也用不到，所以對我來說這足夠了。
　　對要求不是特別高的用戶來說，eOS是一個兼顧實用與美觀的系統，雖然會有小問題，但並不影響，至於不穩定，一般人不會去開服務器，會開服務器的人也不會去用eOS，而且我這段時間使用下來也沒有出現不穩定的情況。
　　總之elementary OS是一個值得你一試的系統。
