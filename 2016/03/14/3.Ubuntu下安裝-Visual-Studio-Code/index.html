Ubuntu下安裝 Visual Studio Code

Mar 14, 2016

　　 昨天在寫eOS的使用體驗的時候，無意間發現VS Code在linux可以用，於是寫完博客後就立馬去安裝VSC，但是照着網上的教程，安裝過程中還是遇到了不少的問題。所以我總結了下，希望能幫到有需要的人。
　　
　　 我的系統是elementary OS，不過是基於Ubuntu的，所以安裝方法相同
　　
　　 Ubuntu Make安裝Visual Studio Code
　　 首先使用PPA方式安裝ubuntu-make

sudo add-apt-repository ppa:ubuntu-desktop/ubuntu-make
sudo apt-get update
sudo apt-get install ubuntu-make
umake web visual-studio-code

　　第四步安裝過程中會讓你確認兩次，第一次是讓你確認安裝目錄，你可以隨意更改，第二次的時候輸入a，回車就行，之後就等待下載安裝完成。如果運氣好的話，安裝按成後啓動器裏應該會出現VSC的圖標。


　　但如果你的臉也和我一樣黑，沒出現的話，沒關係，進入/home/tit1e/.local/share/applications/（系統有區別路徑也可能不一樣，找不到可以直接搜文件夾），你會發現這裏是有圖標的，你可以試着運行它，我發現我無法運行。用編輯器打開它，你會看到如下代碼

[Desktop Entry]
        Version=1.0
        Type=Application
        Name=Visual Studio Code
        Icon=/home/tit1e/visual-studio-code/resources/app/resources/linux/code.png
        Exec=/home/tit1e/visual-studio-code/Code //把Code改成你安裝目錄下這個文件的名字，我目錄下的名字是code，文件名是分大小寫的哦
        Comment=Visual Studio focused on modern web and cloud
        Categories=Development;IDE;
        Terminal=false

　　這時候你再試着運行試試，你會發現可以打開了。
　　至於啓動器的圖標無法顯示，你把這個圖標剪切到其他目錄下，在剪切回來就會顯示了。以上是我的安裝方法。只是這樣安裝的VSC不出現在右鍵運行方式的列表裏，所以打開文檔只能通過VSC內部的open打開。
　　
　　下載壓縮包安裝Visual Studio Code
　　首先去官網下載VSC的zip壓縮包，下載下來後解壓到你想安裝的目錄。
　　然後在啓動器目錄下創建visualstudiocode.desktop文件，名字可以隨意，就是啓動其中顯示的名字。
　　用編輯器打開這個文件，添加如下代碼

[Desktop Entry]
        Version=1.0
        Type=Application
        Name=Visual Studio Code
        Icon=/home/tit1e/visual-studio-code/resources/app/resources/linux/code.png  //圖標路徑，在解壓的文件夾下的/resources/app/resources/linux/目錄下
        Exec=/home/tit1e/visual-studio-code/Code  //解壓目錄下code文件的路徑
        Comment=Visual Studio focused on modern web and cloud
        Categories=Development;IDE;
        Terminal=false

　　接下來保存退出文件，文件夾中就多了一個VSC圖標了，如果圖標不顯示，同樣剪切到其他目錄再剪切回來就OK。
　　請盡情使用吧：)
